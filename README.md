## I. Core Directive: A Living Manifesto
This repository hosts more than just a personal website; it contains a **living digital portfolio**. It is the **ChimeraCore Manifesto**—a public-facing node of `Project Chimera Genesis`.

> **[SYSTEM LOG]:** All previous digital footprints associated with the designation `Bas616` have been deprecated. The entity has evolved into **ChimeraCoreLab**.

## II. System Architecture & Features
This project is a self-contained **vanilla web application**, built with a focus on modern techniques and performance without reliance on heavy frameworks. It rejects modern bloat in favor of raw efficiency and atmosphere.

*   **Single-Page Application (SPA):** Seamless navigation without page reloads.
*   **Modular Data Structure:** Content driven by a separated JavaScript Object for easy updates.
*   **Anti-Aesthetics:** CSS-driven Scanlines, Noise, and Vignette for a "Digital Decay" atmosphere.
*   **Interactive Soundscape:** Integrated audio feedback (SFX) for UI interactions.

## III. Tech Stack

*   **Typography:** IBM Plex Mono, Noto Sans Thai, Onryou, IPAM
